== Description ==

This plugin creates a shortcode so you can embed House to House media on your WordPress site. The shortcode use to embed the HTH media player is as follows: [hth-player media=#####]  The ##### would be replaced by the id for the media file, which can be found at the end of the URL. Just copy the last part of the URL starting at media.

== Installation ==

1. Unzip and upload the folder 'hth-embed-media-player' to your `/wp-content/plugins/` directory
2. Activate 'HTH Embed Media Player' through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= How do I embed HTH Media =
Click on the "Text" tab on your WP editor and insert the the shrotcode([hth-player media=#####]) where you would like the media player to be displayed.
